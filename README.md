# Image Extracter API

An API to extract images from the URL

## Required Tools
- Go 1.21.1+

## Local Setup
```
git clone git@github.com:Ammar022/image-extracter-api.git
```

## Run
```
./scripts/run.sh url destiantion_folder_path
```
